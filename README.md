# Hoops 2K26 - Browser Basketball

3D-Basketballspiel im Browser, gebaut mit Three.js.

Live: https://vmorkovka333-glitch.github.io/stranger-things-game/

## Steuerung
- WASD - Bewegen
- Maus - Kamera
- Linke Maustaste halten / loslassen - Wurf laden und werfen
- SHIFT - Sprint
- SPACE - Springen
- R - Ball zurueckholen
- ESC - Maus freigeben
